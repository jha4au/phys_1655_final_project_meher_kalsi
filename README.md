# phys_1655_final_project_meher_kalsi
Final Project for PHYS 1655

The dataset used in this project examines various lifestyle, dietary, and demographic factors related to obesity levels. It includes variables such as physical activity (FAF), vegetable consumption (FCVC), high-calorie food intake (FAVC), snacking behavior (CAEC), and a categorical measure of obesity (NObeyesdad). The main goal of this analysis was to understand how different health behaviors interact and whether factors like snacking contribute to obesity among individuals with otherwise healthy habits. The results showed that snacking alone was not strongly associated with higher obesity rates. Even among individuals who were physically active, consumed vegetables, and avoided high-calorie foods, those who did not snack actually had a slightly higher obesity rate. Additionally, a regression model found that these behavioral factors explained only a small portion of obesity variation (R² = 0.16), suggesting that obesity is influenced by more complex factors beyond the variables included in this dataset.
